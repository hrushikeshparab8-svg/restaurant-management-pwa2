# Restaurant Management PWA v2

Updated GitHub Pages + Supabase app with:

- Flipkart-style orange/purple mobile UI
- Fast menu-based Sales module
- Expense purchase entry
- Automatic Inventory update from Expenses
- SOP/Recipe costing
- Recipe preparation inventory adjustment
- Dashboard profit calculations
- Reports
- Admin/staff role UI handling

## Important setup

Before uploading the new app files, run `supabase-migration-v2.sql` in Supabase SQL Editor.

## Upload to GitHub Pages

Upload these files to your existing repository root:

- index.html
- style.css
- app.js
- manifest.json
- service-worker.js
- icon-192.png
- icon-512.png

Do not delete your Supabase data.
